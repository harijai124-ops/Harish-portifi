IoT Coal Mine Safety & Health Monitoring System

A real-time safety monitoring system designed for coal mine environments. The system continuously tracks temperature, gas concentration, humidity levels, and worker health metrics using IoT sensors. LCD Display Monitor for Micro Processor

Successfully interfaced a 16×2 LCD display module with a microprocessor to visualize processed data in real time. The project demonstrated register-level programming, data bus communication, display driver implementation, and timing control. Multiple data sets were tested to verify output accuracy and synchronization integrity across different clock speeds.

Project 03 · Power Systems
🔌
Automatic Power Failure Detection System

Designed and built an automatic circuit that continuously monitors power supply lines and detects failures or voltage drops in milliseconds. Upon detection, a relay-based backup switching mechanism activates alongside audio-visual alerts. Tested under multiple load conditions, the circuit demonstrated consistent and reliable detection — suitable for critical infrastructure and industrial safety applications.
